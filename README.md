# Customer Segmentation using SQL and Power BI

## 📌 Overview
This project performs customer segmentation using RFM (Recency, Frequency, Monetary) analysis to identify high-value and at-risk customers.

## 🛠️ Tools Used
- MySQL (SQL)
- Power BI

## 📊 Process
1. Cleaned and analyzed sales data using SQL  
2. Calculated RFM metrics (Recency, Frequency, Monetary)  
3. Segmented customers into:
   - Champions
   - Loyal Customers
   - At Risk
   - Regular Customers  
4. Visualized insights using Power BI  

## 📈 Key Insights
- Champions contribute highest revenue  
- At-risk customers show low engagement  
- Loyal customers provide consistent value  

## 📁 Files
- rfm_analysis.sql → SQL queries  
- customer_segments.csv → processed dataset  
- dashboard.pbix → Power BI dashboard
